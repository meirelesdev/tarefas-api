# Tarefas Api

> API REST para gerenciamento de tarefas desenvolvida em PHP, com rotas para criar, listar, atualizar e deletar tarefas com banco de dados MySQL.

## Stack

- PHP

## Prerequisitos

- PHP 7.4+ e Composer

## Como executar

```bash
# Clone o repositorio
git clone https://github.com/meirelesdev/tarefas-api.git
cd tarefas-api

# Instale as dependencias
composer install

# Configure as variaveis de ambiente
cp .env-example .env
# Edite o arquivo .env com suas configuracoes

# Inicie a aplicacao
php -S localhost:8000
```
